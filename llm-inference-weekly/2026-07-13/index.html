P02026-07-09

SiFAR

Synchronization-Free All-Reduce for Low-Latency LLM Inference

Tensor ParallelAll-ReduceLow Latency

用 Dual Buffering、交换机内归约和 Speculative Reduction 消除 TP 通信屏障，把跨 Decode step 的隐式同步转化为可复用状态。

展开判断
  • 证据：All-Reduce 延迟最高降低 52%；TP=8 端到端吞吐提升 13.1%–18.6%。
  • 适用：多 GPU、低 Batch、TPOT 严格的 reasoning 推理。
  • 边界：依赖 H200 与先进网络；对单卡和大 Batch 服务价值较弱。

P02026-07-10

Hardware-Friendly LLM Design

AI Model Co-Design · NVIDIA Technical Report

Model–System Co-designGEMMNVFP4

将矩阵形状、Tile 对齐、宽深比、低精度与 Attention/FFN 并行方式前置到模型设计，强调少量大算子优于大量碎片算子。

展开判断
  • 可迁移：接近方形的大矩阵、维度对齐、适度宽而浅、Attention/FFN 解耦并行。
  • 与你的连接：本质上是 RankMixer“碎片算子 → 大 GEMM”在 LLM 推理上的延伸。
  • 边界：256/512 对齐和 NVFP4 数据主要针对 Blackwell，不能机械套到 A10。

P02026-07-09

System-Aware KV Cache Survey

Towards Efficient Large Language Model Serving

KV InfrastructureACL 2026Survey

用 Temporal、Spatial、Structural 三个系统行为维度重构 KV Cache 研究地图，把压缩、迁移、调度与保留连接到具体 SLO。

展开判断
  • 价值：适合建立 KV 表示—放置—迁移—调度—保留的完整知识树。
  • 核心反直觉：压缩率高不等于端到端快；命中率高也可能被回迁带宽抵消。
  • 定位：不是单一新算法，而是当前最实用的系统化入口。

P12026-07-07

Akashic

A Low-Overhead LLM Inference Service with MemAttention

Agent MemoryChunk RetrievalPlacement

将长期历史组织为有界 Chunk，同时联合优化语义关系与物理共置，减少完整历史重放和碎片化 I/O。

展开判断
  • 证据：吞吐最高 1.21×，可持续请求率最高 1.88×。
  • 迁移价值：经常共同检索的历史片段，是否也应在存储侧共同放置。
  • 边界：更接近 Memory/RAG 系统，不是原生 Transformer KV 压缩。

P12026-07-08

SGLang HiCache + X-Mem

A Dedicated High-Bandwidth Memory Tier for Prefix KV

Prefix CacheMemory TieringTTFT

当 Prefix 命中率接近 98% 后，瓶颈从“能否复用”转向“能否足够快地把 KV 搬回 HBM”。

展开判断
  • 证据：对比 Host DRAM，单请求 TTFT 最高改善约 6.7×。
  • 评估账本：Hit Rate、Bytes Fetched、Fetch Bandwidth、TTFT Saved 必须同时记录。
  • 边界：供应商报告；依赖超高前缀命中率和专用外部内存。

P12026-07-03

SPORK

Self-Speculative Forking to Accelerate Agentic LLM Inference

SpeculationTool LatencyP95

让模型提前预测即将调用的工具，使工具执行与剩余推理 Token 重叠，把优化范围扩展到模型外部的完整请求链路。

展开判断
  • 证据：Qwen3-32B GAIA P95 从 131.9 秒下降到 108.1 秒。
  • 启发：请求延迟不只是 Prefill + Decode，还包含工具、检索、CPU、网络与队列。
  • 边界：收益高度依赖工具等待占比，对普通聊天推理不成立。

P22026-07-04

SLO-Aware Online Routing

Online Linear Programming for Multi-Objective Routing

SchedulerSLOOnline LP

用动态 Shadow Price 在 KV 容量、TTFT、吞吐与尾延迟之间做显式决策，尝试替代 queue-length 等单一启发式规则。

展开判断
  • 方向价值：调度器从启发式规则走向可解释多目标优化。
  • 工程要求：毫秒级更新 shadow price，必须证明决策开销稳定。
  • 边界：当前主要基于 Vidur 模拟器，生产证据不足。
